Chaînes de caractères

Une chaîne de caractères est simplement un tableau de type char.

Les fonctions standard de traitement des chaînes de caractères supposent que les caractères significatifs d’une chaîne se situent avant le caractère butée '\0', qui devient donc de ce fait obligatoire.

char s1[] = { 'H','e','l','l','o'} ; // s1 contient 5 caractères H,e,l,l,o
char s2[] = "Hello" ;                // s2 contient 6 caractères H,e,l,l,o,\0
fonctions à connaitre : strcpy, strcmp, strlen, strcat

1. Exemple 1

Exemple d’utilisation des fonctions standard de manipulation de string
#include <stdio.h>
#include <string.h>

int main()
{
  char s1[10], s2[10], x[10] ;

  /* lire 2 chaînes au clavier */
  fgets(s1,10,stdin) ;
  fgets(s2,10,stdin) ;

  if (strcmp(s1,s2)>0)
  {
    strcpy(x,s1) ;
    strcpy(s1,s2) ;
    strcpy(s2,x) ;
  }

  /* ecrire les valeurs de s1 et s2 */
  printf("%s\n%s\n",s1,s2) ;

  return 0 ;
}
man 3 string donne toutes fonctions de traitement des chaînes de caractères existantes dans la bibliothèque standard C.

2. Exemple 2

Exemple de manipulation d’une string caractère par caractère
#include <stdio.h>

#define L 20

int un_a_un(char *s)
{
  int i=0 ;

  while (s[i]!='\0') i++ ;

  return i ;
}

int main()
{
  char s[L] ;

  fgets(s,L-1,stdin) ;
  printf("%d\n",un_a_un(s)) ;

  return 0 ;
}
